FastCap2Object.Stop

void FastCap2Object.Stop()

Return Value

None

Parameters

None

Remarks

Stops the currently running simulation.

Notes:

It could take some time before FastCap2 actually stops, depending on the operation currently under execution. You can poll FastCap2 status checking the IsRunning property or you can set a callback upon FastCap2 end of operation using the SetEndCallback method.
